Given a positive integer num,
return the next greater integer
that can be made from its digits.
If such a number doesn't exist,
return num itself

```
input: num = 3401   
output: 3410
```

```
input: num = 1318540
output: 1340158
```

```
input: num = 3410   
output: 4013
```
